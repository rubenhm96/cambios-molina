**Manage, track and report the advertising management campaigns and banners on your WordPress site. Turn your WP into an Ad Server!**

The CM Ad Changer manages several banner advertising campaigns across multiple WordPress sites simultaneously. The plugin allows for an unlimited number of banners and ad campaigns to be created and used.

As a WordPress ad management server, the plugin settings manage or restrict the appearance of banner ads according to each campaigns parameters. 

This customizable Wordpress ad manager plugin has settings that include days of the week, custom ad campaign dates, and in pro version maximum numbers of impressions or clicks per banner. 

[vimeo https://vimeo.com/108688776]

The ad management pro plugin customizations include mobile responsive ads, popup ads, html ads, video ads and banner ad rotation.  Users can also choose the advert type, determine the weight of each individual banner and choose optimal times to run ad campaigns. 

This WordPress advertising plugin lets ads be displayed within WordPress templates or using simple shortcodes. 

The remote client plugin can connected to an ad manager installed on another WordPress site. The ad management plugin also features the ability to automatically re-size banners based on the device the user is using. 

The ad-server's management console is intuitive and very user friendly. Users can easily view statistics for each campaign and track their progress on multiple WordPress websites.

== Basic Features == 

* Includes impression &amp; clicks count per each banner [view image](http://ad-changer.cminds.com/wp-content/uploads/ad-changer/image-a1.png)
* Each campaign can serve banners randomly or selectively.
* Each banner can has it's own weight allowing it to have more impressions when selected randomly [view image](http://ad-changer.cminds.com/wp-content/uploads/ad-changer/image-a1.png)
* Each banner can have a unique target URL [view image](http://ad-changer.cminds.com/wp-content/uploads/ad-changer/image-a1.png)
* Ads are inserted into post / pages using shortcode.
* Shortcode support debug mode [view image](http://ad-changer.cminds.com/wp-content/uploads/ad-changer/image7.png)
* Shortcode also includes ability to wrap banner with div.
* Can run several campaigns simultaneously [view image](http://ad-changer.cminds.com/wp-content/uploads/ad-changer/image2.png)
* Each campaign can manage unlimited number of images / banners.
